## 产品 / API 动态

### OpenAI

#### Our response to the Axios developer tool compromise
*发布：2026-04-10 · 类型：产品/公告 · 分类：product-api*

**发布内容：** OpenAI 披露一次与 Axios npm 包被投毒相关的软件供应链事件。受影响的是 macOS 应用签名流程中的 GitHub Actions workflow，该流程曾下载并执行被篡改的 Axios 1.14.1。该 workflow 可访问用于签名 ChatGPT Desktop、Codex、Codex CLI 和 Atlas 的 macOS code‑signing certificate。OpenAI 未发现证书或用户数据被成功窃取，但出于安全考虑已将该证书视为可能泄露并进行吊销与轮换。

**关键措施：**
- 旋转并重新签发 macOS code‑signing certificate
- 重新发布所有 macOS 客户端版本
- 与 Apple 协作阻止旧证书的新 notarization
- 审计所有历史 notarization 记录，确认没有异常软件

**用户影响：**
- 仅影响 macOS 桌面应用
- Web、iOS、Android、Linux、Windows 不受影响
- 未发现用户数据或 API key 泄露

**重要时间点：**
- 2026‑03‑31：Axios 供应链攻击被发现
- 2026‑05‑08：旧证书将正式吊销，旧版本 macOS 应用可能无法继续运行

**需要更新的最早安全版本：**
- ChatGPT Desktop: 1.2026.051
- Codex App: 26.406.40811
- Codex CLI: 0.119.0
- Atlas: 1.2026.84.2

**适用场景：** macOS 用户需要更新 OpenAI 客户端，以确保应用由新的官方证书签名并避免潜在伪造软件风险。
