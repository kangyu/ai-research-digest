## 安全与对齐

### OpenAI

#### Introducing the OpenAI Safety Fellowship
*发布：2026-04-06 · 类型：产品/公告 · 分类：safety*

**发布内容：** OpenAI 启动 Safety Fellowship 计划，面向外部研究者资助 AI safety 与 alignment 相关研究。

**关键信息：**
- 项目周期：2026‑09‑14 → 2027‑02‑05
- 提供 stipend、API credits 与计算资源
- 研究方向：safety evaluation、robustness、agent oversight、privacy‑preserving safety、misuse mitigation 等

**适用对象：** 独立研究者、工程师或跨学科研究人员（CS、cybersecurity、HCI、social science 等）。

**目标：** 产出论文、benchmark、dataset 等可公开的研究成果，并推动安全研究社区发展。

## Agent 开发

### Anthropic

#### Claude Opus 4.6
*发布：2026-04-07 · 类型：研究/模型 · 分类：agent-dev*

**核心贡献：** Claude Opus 4.6 引入 1M token 长上下文（beta），强化 agentic coding、长任务规划和复杂推理能力。

**方法与能力：**
- 更强的多步骤任务规划与工具调用
- 在大型代码库中的调试与代码审查能力显著提升
- 支持 agent teams 与长时间 autonomous workflows

**实验结果：**
- Terminal‑Bench 2.0 agent coding 评测排名第一
- Humanity’s Last Exam 多学科推理测试领先
- GDPval‑AA 知识工作任务评测领先约 144 Elo

**意义：** 强化长上下文与 agent workflow 能力，面向复杂软件工程与知识工作自动化场景。
